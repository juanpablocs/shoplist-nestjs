# App Store

App basic shop store created with Nest, actions created:

- Auth login
- Auth register
- List products
- Create product
- List cart
- Add cart
- List orders
- Create order

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
